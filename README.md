# Shipping Calculator Pro

A robust shipping calculator web application for logistics and supply chain management.

## Description

This application helps calculate shipping costs and provides delivery estimates to customers, enabling transparent and efficient supply chain operations.

## Features

- Calculate shipping costs based on distance, weight, and delivery time
- Provide accurate delivery estimates
- Support for multiple shipping methods
- Simple interest calculation for additional fees

## Installation
```bash
git clone https://github.com/naanvaran2003-hash/shipping-calculator-Pro.git
cd shipping-calculator-Pro
```

## Usage

Run the shipping cost calculator:
```bash
bash simple-interest.sh
```

The calculator will prompt you for:
- Principal amount (base shipping cost)
- Rate of interest (additional fee percentage)
- Time period (number of days)
